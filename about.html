the bucket webring was created in january 2022 and is currently managed by bucketfish.

for issues or bugs or suggestions, open a pr on the github repo or find us on discord!

joining

although our members are based on discord, there is no prerequisite for joining. submit a pr to this repo by modifying webring.json.

simply add your site's info to the bottom of the json by adding a new field:

{ "name": "<name>", "url": "<url>" }

  • <name> is your name to be displayed on this website :)
  • <url> is the url to your webpage.

after your pr is approved, please add the navigation bar to your webpage using an embed - which will be explained below!


alternatively, if you don't know how json or github works, you can also shoot me a message at bucketfish#3961 on discord and i'll add you in.

embedding

you're highly encouraged to add this navbar to your site to allow users to navigate through the webring.

add the following iframe to your site, and replace YOURNAME with the name that you registered using:

<iframe id="bucket-webring" style="width: 100%; height: 3rem;" src="https://webring.bucketfish.me/?name=YOURNAME"></iframe>
